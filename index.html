Advocacy Toolkit

What does advocacy mean?

Advocacy is when we speak up for our rights.

Advocates work to:

  • Make sure everyone gets the same chances to work, eat, sleep, and live their lives.
  • Make sure everyone feels safe.
  • Make sure everyone is healthy.
  • Make sure everyone is heard.
  • Make sure that our land stays clean and safe.
  • Make sure that everyone has the same information.

Anyone can be an advocate for themselves or for someone else.
